==========================
delete_os_tenant_resources
==========================


Description
-----------

This is a script to list/delete the resources of a tenant (and also the keypairs of the user)

It uses the OpenStack Pythons API:

* python-keystoneclient
* python-novaclient
* python-glanceclient
* python-cinderclient
* python-neutronclient
